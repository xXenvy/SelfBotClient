Tasks
======

A :class:`SelfBotClient.tasks` function allows you to run multiple tasks simultaneously
---------------------------
(resulting in faster ratelimit), to take advantage of this you need to change the argument **use_threading=True**
in the :class:`SelfBotClient.client.Client` params
